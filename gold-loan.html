Smart Gold Loan
Management System.

Gold loans represent one of the most critical and fast-moving credit assets in non-banking financial companies (NBFCs). At Artcode Private Limited, I engineered the core backend operations for a multi-branch, multi-role Gold Loan management SaaS platform designed to transition traditional offline lending activities into highly secure digital pipelines.

This system handles the end-to-end lifecycle of a loan, beginning with lead capture and credit approvals, transitioning to dynamic ROI scheme calculations, automated ledger allocations, real-time EMI tracking with automated alert triggers, and final loan closure workflows. Built using Spring Boot, Hibernate/JPA, and MySQL, the backend guarantees transaction isolation and auditing compliance under rapid credit processing cycles.

The Engineering Challenge

Lending operations demand absolute transaction accuracy and absolute compliance. Calculating dynamic interest structures (reducing balance, bullet repayment, or step-up interest rates) based on daily gold rate variables requires absolute precision, where a rounding error of 0.01% could lead to heavy reconciliation deficits in bulk daily audits.

Furthermore, financial regulators mandate a strict audit trail to prevent collusion. Operations such as changing interest rates, granting waivers, or dispersing large credit limits cannot be processed by a single administrator. The system must enforce a secure **Maker-Checker dual authorization loop** natively in the API level to maintain financial auditing standards.

My Implementation Role

I engineered the transaction ledger algorithms, mapped database entity relationships under strict transaction isolation levels, and implemented the dynamic loan math libraries.

  • Dynamic ROI & Scheme Calculator

    I designed a metadata-driven Scheme Configurator engine. Instead of hardcoding loan interest rules, branch managers configure rate parameters (reducing, compounding, daily accrual schedules) dynamically, which are parsed and processed by my backend math library using high-precision Java BigDecimal classes.

  • Maker-Checker Authorization Core

    I implemented a secure Maker-Checker approval workflow framework. Critical operations (loan approvals, waiver grants) are logged to a 'Pending Approval' state ledger by a 'Maker' agent. An encrypted transaction token is generated and published, which is only validated and committed upon signature from an authorized 'Checker' officer.

  • Automated EMI Scheduler & Alert Engine

    I engineered an automated scheduler using Spring Batch that runs bulk daily checks across active accounts. The scheduler computes due EMI allocations, logs default alerts in MySQL indices, and triggers asynchronous communication webhooks to notify clients of upcoming or overdue bills.

  • Audited Transaction Ledger

    I mapped a highly normalized ledger schema utilizing Hibernate/JPA transaction bindings. By configuring strict `@Transactional(isolation = Isolation.REPEATABLE_READ)` isolation boundaries, I guaranteed that parallel repayments and interest accruals occur without causing race conditions or dirty database reads.
